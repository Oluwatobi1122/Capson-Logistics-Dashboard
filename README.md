Logistics Dashboard — Power BI Project

 Overview
A full logistics database analysis built as part of my 
Data Analytics course at TS Academy.

 Dataset
14 operational tables covering:
- 85,410 loads delivered
- 150 drivers | 120 trucks | 200 customers
- 3 years of data (2022 - 2024)

 Dashboards Built
1. Revenue Overview — $262.5M total revenue
2. Driver Performance — OTD rate analysis
3. Fleet and Maintenance — utilisation and cost breakdown
4. Safety and Fuel — incident claims and fuel trends

 Tools Used
- Microsoft Excel (data joining and cleaning)
- Power BI (dashboard and DAX measures)

 Key Findings
- Total Revenue: $262.5M across 3 years
- Fleet Utilisation: 83% average
- Profit: $158.5M
- 37.6% of safety incidents were preventable
